# Данные

Данные в git НЕ хранятся (большие). Структура:

- `raw/` — исходный корпус (папка «Источники информации»). Положи сюда документы.
- `processed/` — артефакты пайплайна: `documents.jsonl`, `graph.pkl`, индекс.
- `sample/` — маленький набор (3-5 документов) для быстрой проверки «из коробки».

Полные обработанные артефакты выкладываются на облачный диск (ссылка в материалах сдачи).
